1. 아나콘다 가상환경 생성(파이썬 버전 >=3.8)
2. git clone repo
3. 리콰이어먼트.txt 설치
4. cpu,gpu 사용할 device를 333.py에서 설정
5. 터미널에서 333.py 실행
